Quick Start

1. Go to the Creator tab to start writing.

2. Go to the Library tab to read your "Featured Stories" (from your GitHub `stories/` folder) and your own "Local Creations."

3. Go to the Settings tab to export your creations. To publish, you must manually create a new `.json` file in your GitHub `stories/` folder.
